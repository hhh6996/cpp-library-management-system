### 阶段 3：代码讲解（3-4 分钟）

```
Alt+Tab 切回 VS2022

══════════ 讲解 1：继承与虚函数 ══════════

1. 打开 User.h，鼠标指向第 15-18 行：
   "四个纯虚函数 = 0 ——所有派生类必须实现"

2. 打开 Student.cpp，指向第 22-26 行：
   "getMaxBorrow 返回 5，getBorrowDays 返回 30"

3. 打开 Teacher.cpp，指向第 22-26 行：
   "同样的函数，Teacher 返回 10 和 60"

══════════ 讲解 2：多态 ══════════

4. 打开 Library.cpp，Ctrl+G 跳到第 331 行：
   "borrowItem 函数——理解多态的最佳入口"

5. 指向第 353 行 u->getMaxBorrow()：
   "findUser 返回 User*，不知道具体类型
    但 getMaxBorrow() ——Student 自动返回 5，Teacher 返回 10
    Library 不需要 if-else 判断——这就是多态"

6. 指向第 373 行 u->getBorrowDays()：
   "借阅天数也是多态——Student 30 天，Teacher 60 天"

══════════ 踩坑 ══════════

7. 跳到 deleteUser 函数（约第 273 行）：
   "最初删除读者时只检查了未归还记录就放行
    但已归还的历史记录还指着被删的 User——悬空指针，后面查记录就崩溃
    修复：第 283 行——删除前检查所有关联记录，有就拒绝删除"
```
